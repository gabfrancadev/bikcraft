# Bikcraft

Bikcraft is a complete multi-page website developed as part of the Origamid front-end course.  
The project simulates a real product website and focuses on building solid foundations in HTML, CSS, and JavaScript.

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## Project Structure
- Multiple pages (Home, Products, Insurance, Contact, etc.)
- Organized CSS structure
- JavaScript used for interactivity and UI behavior

## What I Learned
- Semantic HTML and accessibility fundamentals
- CSS layout techniques (Flexbox and Grid)
- Responsive layout concepts
- JavaScript DOM manipulation
- Structuring a real-world front-end project
- Version control with Git and GitHub
